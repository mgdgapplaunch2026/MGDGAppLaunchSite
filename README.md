## 02. Technical Stack

The system is built on a "Privacy-First" stack, ensuring institutional-grade reliability.



* **Frontend:** `Tailwind CSS` / `Vanilla JS` (Institutional Framework)

* **Engine:** `Node.js` / `Express Runtime`

## 03. Environment Configuration

To run the protocol locally, you must initialize the following environment variables in a `.env` file. 



> [!IMPORTANT]

> Never commit the actual `.env` file to version control.



```env

PORT=3000

MONGODB_URI=your_mongodb_connection_string

RESEND_API_KEY=your_resend_api_key

NODE_ENV=development



```
